# 🎓 Job Placement Data Analysis

A Python-based Data Analysis project that explores the factors affecting student placements using academic records, employability test scores, work experience, and specialization data.

## 📌 Project Overview

The objective of this project is to analyze a job placement dataset and identify patterns that influence student placement outcomes. The project performs data exploration, statistical analysis, and data visualization using Python libraries.

## 🚀 Features

* Dataset exploration and summary statistics
* Missing value analysis
* Academic performance analysis
* Placement percentage calculation
* Specialization-wise placement analysis
* Work experience analysis
* Employability test score distribution
* Correlation analysis using heatmaps
* Graphical representation of important insights

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## ▶️ Installation

### Clone the Repository

```bash
git clone https://github.com/virajkotawadekar/Campus-Placement-Data-Analysis.git
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Project

```bash
python job_placement_analysis.py
```

## 📈 Key Insights

* Higher academic performance generally improves placement opportunities.
* Students with work experience tend to have better placement outcomes.
* Employability test scores show a positive relationship with placements.
* Specialization plays an important role in determining placement success.
